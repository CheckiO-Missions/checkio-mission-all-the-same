З чого мені розпочати вирішувати цю місію?
Ти можеш підрахувати кількість першого елемента в списку, і якщо отримана кількість і загальна кількість елементів (довжина списку) однакові, то всі елементи списку дійсно однакові. Також не забудь врахувати особливий випадок - пустий список, який за умовами завдання також повинен повертати True.
Не міг би ти уточнити деталі?
Все досить просто. Потрібно підрахувати кількість входжень першого елемента в списку. Наприклад, за допомогою методу list.count(x). Про це та інші методи роботи зі списком ти можеш прочитати в official Python documentation on lists. Він працює наступним чином: 
elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
elements.count('a')
elements.count('a') --> 3
Потім потрібно порівняти це число з довжиною списку. В цьому тобі допоможе ця інформація official documentation on the len() function. Якщо ці значення рівні, то список складається тільки з однакових елементів.
А якщо я тільки знаю, що потрібний елемент знаходиться на першій позиції?
Якщо нам потрібен елемент у певній позиції - ми можемо звернутися до нього через індекс:
elements.count(elements[0])
Позиція з індексом 0 - перша позиція в списку.
Чудово. Але що робити з даними про кількість першого елемента?
Тепер потрібно порівняти цю інформацію з довжиною вхідного списку (загальною кількістю елементів). Якщо ми повернемося до першого прикладу: 
elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
elements.count(elements[0])
і дізнаємося, чи дорівнює кількість першого елемента загальній кількості елементів у списку, тоді ми дізнаємося, чи складається список з усіх однакових елементів чи ні.
elements.count('a') == len(elements) --> False, оскільки номер першого елемента дорівнює 3, а загальна кількість елементів у списку дорівнює 8. В іншій ситуації: 
elements = ['a', 'a', 'a', 'a']
elements.count(elements[0])
elements.count(elements[0]) == len(elements) --> Правда, оскільки ці значення (номер першого елемента і число всіх елементів списку) - збігаються.
Як записати це в коді?
Твоя функція виглядатиме так: 
def all_the_same(elements):
    if elements.count(elements[0]) == len(elements):
        return True
    return False
Той самий код можна переписати більш елегантним способом: 
def all_the_same(elements):
    return elements.count(elements[0]) == len(elements)
Що робити зі списком, якщо в ньому немає елементів?
У цьому випадку ми також припускаємо, що всі елементи однакові, і твоя функція має повертати True.
Як буде виглядати функція з урахуванням цієї інформації?
Якщо зібрати все разом, то отримаємо: 
def all_the_same(elements):
    if elements == []:
        return True
    return elements.count(elements[0]) == len(elements)
