Всё достаточно просто. Вам необходимо посчитать количество вхождений первого элемента в список. Например, используя метод array.count(x) Он работает следующим образом: 
        elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
        elements.count('a')
elements.count('a') --> 3
Или, если нам нужен элемент, находящийся на определенной позиции - можно обращаться по индексу:
        elements.count(elements[0])
Затем нужно сравнить это число с длиной списка. Если эти значения равны - значит список состоит только из одинаковых елементов.

Также вам необходимо убедиться, что длина исходного списка больше 1. Так как если список состоит только из 1 или 0 элементов - ваша функция должна возвращать True. Это может выглядеть так: 
        def all_the_same(elements):
            if len(elements) <= 1:
                return True
Отлично. А что делать с данными о количестве первого элемента?
Теперь вам необходимо сравнить эту информацию с длиной исходного списка (общим количеством элементов). Если мы вернемся к первому примеру: 
        elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
        elements.count(elements[0])
и узнаем, равно ли количество первого элемента общему количеству элементов в списке, то узнаем, состоит ли список из всех одинаковых элементов или нет.
elements.count('a') == len(elements) --> False, так как количество первого элемента = 3, а всего элементов в списке 8. В другой ситуации: 
        elements = ['a', 'a', 'a', 'a']
        elements.count(elements[0])
elements.count(elements[0]) == len(elements) --> True, так как эти значения (количество первого элемента и количество элементов в списке вообще - совпадают)

Если сложить всё вместе, мы получим: 
        def all_the_same(elements):
